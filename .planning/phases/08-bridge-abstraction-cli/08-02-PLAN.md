  <action>
Create `crates/bm/src/commands/bridge.rs` with handler functions for all 10 bridge subcommands. Each handler follows the same pattern:

1. Load config via `config::load()`, resolve team via `config::resolve_team()`
2. Check `just` is available via `which::which("just")` — bail with "Bridge commands require 'just'. Install it: https://just.systems/" if missing
3. Discover bridge via `bridge::discover(&team_repo, &team.name)` — if None, print "No bridge configured for team '{name}'." and return Ok(())
4. Load manifest via `bridge::load_manifest(&bridge_dir)`
5. Execute the specific operation

**Handlers:**

`start(team: Option<&str>) -> Result<()>`:
- Check bridge_type != "external" (print "Bridge '{name}' is external -- lifecycle commands are not available. The service is managed externally." and return Ok)
- Get lifecycle commands from manifest (bail if None for local bridge)
- invoke_recipe for lifecycle.start
- invoke_recipe for lifecycle.health
- Load state, update: bridge_name, bridge_type, status="running", started_at=now (chrono::Utc::now().to_rfc3339()), service_url from config exchange url field, last_health_check=now
- Save state
- Print "Bridge '{name}' started."

`stop(team: Option<&str>) -> Result<()>`:
- Same external bridge check as start
- invoke_recipe for lifecycle.stop
- Load state, update: status="stopped", clear started_at
- Save state
- Print "Bridge '{name}' stopped."

`status(team: Option<&str>) -> Result<()>`:
- Load state from state_path
- If bridge_name is None, print "No bridge active for team '{name}'." and return
- Print bridge info: name, type, status, URL (if present), started_at (if present)
- If identities non-empty, display in comfy-table with columns: Username, User ID, Created
- If rooms non-empty, display in comfy-table with columns: Room, Room ID, Created
- Use `comfy_table::presets::UTF8_FULL_CONDENSED` and `modifiers::UTF8_ROUND_CORNERS` (same as status.rs)

`identity_add(username: &str, team: Option<&str>) -> Result<()>`:
- invoke_recipe for identity.onboard with args [username]
- Parse config exchange: extract username, user_id, token fields
- Load state, add BridgeIdentity to identities HashMap keyed by username, set bridge_name/bridge_type if not set
- Save state
- Print "Identity '{username}' added to bridge '{name}'."

`identity_rotate(username: &str, team: Option<&str>) -> Result<()>`:
- Check username exists in state.identities — bail with "Identity '{username}' not found. Run 'bm bridge identity list' to see registered identities."
- invoke_recipe for identity.rotate_credentials with args [username]
- Parse config exchange: update token (and user_id if changed)
- Save state
- Print "Credentials rotated for '{username}'."

`identity_remove(username: &str, team: Option<&str>) -> Result<()>`:
- Check username exists in state.identities
- invoke_recipe for identity.remove with args [username]
- Remove from state.identities
- Save state
- Print "Identity '{username}' removed."

`identity_list(team: Option<&str>) -> Result<()>`:
- Load state
- If identities empty, print "No identities registered."
- Otherwise, display in comfy-table: Username, User ID, Created

`room_create(name: &str, team: Option<&str>) -> Result<()>`:
- Check manifest has room spec — bail with "Bridge '{name}' does not support room management."
- invoke_recipe for room.create with args [name]
- Parse config exchange: extract name, room_id
- Add BridgeRoom to state.rooms
- Save state
- Print "Room '{name}' created."

`room_list(team: Option<&str>) -> Result<()>`:
- Check manifest has room spec
- invoke_recipe for room.list
- Parse config exchange: extract rooms array
- Display in comfy-table: Room, Room ID

```rust
Command::Bridge { command } => match command {
    BridgeCommand::Start { team } => commands::bridge::start(team.as_deref())?,

    BridgeCommand::Status { team } => commands::bridge::status(team.as_deref())?,
    BridgeCommand::Identity { command } => match command {
        BridgeIdentityCommand::Add { username, team } => commands::bridge::identity_add(&username, team.as_deref())?,
        BridgeIdentityCommand::Rotate { username, team } => commands::bridge::identity_rotate(&username, team.as_deref())?,
        BridgeIdentityCommand::Remove { username, team } => commands::bridge::identity_remove(&username, team.as_deref())?,
        BridgeIdentityCommand::List { team } => commands::bridge::identity_list(team.as_deref())?,
    },
    BridgeCommand::Room { command } => match command {
        BridgeRoomCommand::Create { name, team } => commands::bridge::room_create(&name, team.as_deref())?,
        BridgeRoomCommand::List { team } => commands::bridge::room_list(team.as_deref())?,
    },
},
```
