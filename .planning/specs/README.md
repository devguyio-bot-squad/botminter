# Specifications

This directory contains formal specifications for BotMinter's external contracts and plugin interfaces. Specs define the "what" -- the shape of a contract that third-party implementors must follow.

For the discipline governing how specs are written, see [meta-spec.md](meta-spec.md).

For architectural decision rationale (the "why"), see [ADRs](../adrs/).
